# HGSC_RNAseq

This repository will in the near future be populated with the code necessary to recreate all analyses in the study "Transcriptomic profiling of paired primary tumour and omental metastases in High-Grade Serous Ovarian Carcinoma highlights site-dependent expression patterns and neoadjuvant chemotherapy-associated changes" Vanagas et al. 2026. Part of the Advanced Cancer Biology Course, Faculty of Medicine, Lund University, Sweden.
